SEAL is an autonomous task runner for tech leads. Send a WhatsApp message or email — SEAL structures it, schedules it, and executes it via AI.

💬

WhatsApp & Email Capture

Send tasks from your phone. Text, voice notes, or email — SEAL ingests everything and creates structured tasks.

🎧

Voice Transcription

Record a voice note on WhatsApp. SEAL transcribes it locally via Whisper and turns it into an actionable task.

⚙️

Autonomous Execution

Tasks run automatically via Claude. Code reviews, test runs, deployments — SEAL spawns AI sessions and reports back.

🔔

Unignorable Reminders

5 notification levels from silent to supernova. Supernova re-fires every 5 minutes until you acknowledge it.

💻

Multi-Computer Sync

Local SQLite by default. Add two environment variables to sync tasks across all your machines via Turso.

📚

Project-Aware

Mention a project name and SEAL routes it automatically. No project? SEAL asks you which one via WhatsApp reply.

How it works

1

Capture

Send a message to SEAL via WhatsApp, email, voice note, or the /seal command in Claude Code.

2

Structure

SEAL detects the project, type, priority, and schedule. Voice notes are transcribed automatically.

3

Execute

Tasks run autonomously via Claude. Up to 4 parallel AI sessions. Results saved and notifications sent.

Examples

# WhatsApp messages
valenty: run all tests
fix the auth bug on nanoclaw
remind me to review PR by Friday

# Email to seal@hens.com.br
Subject: deploy hotfix on mage

# Voice note on WhatsApp
"Check if the CI pipeline is passing on valenty"
→ transcribed → task created → executed
